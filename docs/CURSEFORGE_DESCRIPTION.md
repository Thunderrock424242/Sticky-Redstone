# Sticky Redstone

Bring your redstone wiring off the floor.

**Sticky Redstone** adds a new redstone wire variant that can be placed on **floors, walls, and ceilings**. It keeps vanilla-style signal behavior while making compact builds, vertical wiring, and cleaner contraptions much easier.

---

## Features

- **Surface placement in every direction**  
  Place Sticky Redstone Wires on floor, wall, or ceiling faces.

- **Sticky survival**  
  Sticky redstone clings to surfaces and does not pop off when support changes.

- **Vanilla dust compatibility (configurable)**  
  Optional behavior lets normal redstone dust place sticky wire on walls/ceilings too.

- **Redstone-friendly connectivity**  
  Works with vanilla redstone components and connections as expected.

---

## Crafting

**Sticky Redstone Wire**

- 1x Slime Ball
- 1x Redstone Dust

(Shaped recipe: Slime Ball above Redstone Dust)

---

## Config

`vanillaDustWallsAndCeilings` (default: `false`)

- `true`: Using vanilla redstone dust can place sticky wire on floor/wall/ceiling surfaces.
- `false`: Only the Sticky Redstone Wire item places the block.
